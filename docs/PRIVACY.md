## 2. Information We Collect

### 2.1 Personal Information

We collect information that you voluntarily provide to us when you:

- **Register for an account:** Email address, display name, password (hashed)
- **Update your profile:** Profile picture, bio, avatar preferences
- **Use OAuth authentication:** Information from your Google account (email, name, profile picture) when you choose to sign in with Google

### 2.2 Content Data

When you use IdeaBoard, we collect and store:

- **Stories and Boards:** All content you create including stories, boards, notes, drawings, and connections
- **Components:** Variables and data you define within your projects
- **Attachments:** Images and files you upload to your notes
- **User Preferences:** Application settings, theme choices, and workspace configurations

### 2.3 Automatically Collected Information

We may automatically collect certain information when you access IdeaBoard:

- **Device Information:** Browser type, operating system, device type
- **Usage Data:** Features used, interactions with the application, session duration
- **Log Data:** IP address, access times, pages viewed, referring URL
- **Cookies and Similar Technologies:** Session tokens, authentication cookies

## 4. Data Storage and Security

### 4.1 Storage Location

Your data is stored securely using **Supabase** infrastructure, which utilizes:

- PostgreSQL databases with encryption at rest
- Secure cloud storage for file attachments
- Data centers with industry-standard physical security

### 4.2 Security Measures

We implement appropriate technical and organizational measures including:

- **Encryption:** Data encrypted in transit (TLS/SSL) and at rest
- **Authentication:** Secure password hashing, JWT tokens with rotation
- **Access Control:** Row Level Security (RLS) policies ensuring users can only access their own data
- **Regular Backups:** Automated database backups for disaster recovery

### 4.3 Data Retention

- **Active Accounts:** We retain your data as long as your account is active
- **Deleted Content:** When you delete content, it is removed from our active databases
- **Account Deletion:** Upon account deletion, we remove your personal data within 30 days, except where retention is required by law
